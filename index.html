Tecnologias que eu utilizo

HTML 5 é uma linguagem usada para criar e estruturar páginas da web. Ele define os conteúdos básicos, como textos, imagens e links, permitindo a construção de interfaces web.

CSS3 é uma linguagem de estilo que define a apresentação visual das páginas web. Ele manipula layouts, cores, fontes e outros aspectos de design.

javaScript é uma linguagem de programaçãousada para tornar aplicações web interativas e dinâmicas. Ele permite a manipulação do conteúdo html, resposta a comandos dos usuários, e criação de funcionalidades avançadas.

React Js é uma poderosa biblioteca para construção de interfaces visuais em JavaScript. Ou seja, você pode adotar estruturas do HTML e do JavaScript em um só código, a fim de manipular os aspectos da apresentação de um site.

Github é uma “rede social dev” em que é possível armazenar e compartilhar projetos de desenvolvimento de software. Já o Git é um sistema de controle de versão de arquivos.
